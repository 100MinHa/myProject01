## 이 프로그램은 MBTI 검사를 하는 콘솔 프로그램입니다.
# 💻 프로그램 사용 방법
1. 위의 파이썬 코드를 복사하여 .py 파일로 저장합니다 (예: mbti_quiz.py).

2. 명령 프롬프트나 터미널에서 해당 파일이 있는 위치로 이동합니다.

3. 다음 명령어를 입력하여 프로그램을 실행합니다: python mbti_quiz.py

4. 화면에 나오는 12개의 질문에 대해 '예' 또는 **'아니오'**로 답변을 입력하고 Enter를 누릅니다.

5. 모든 질문에 답하면 MBTI 유형과 해당 유형에 대한 간략한 해설이 출력됩니다.
